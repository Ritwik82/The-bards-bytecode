The Bard's Bytecode

A Shakespeare Programming Language challenge for LNMIIT CTF.

Story. Somewhere between old plays and new compilers, a bored engineer decided to hide a flag inside a Shakespeare play that computers can run. The result is a script where characters shout insults, compare each other to pigs and cats, and occasionally speak their mind as numbers.

Your task is simple: stage the play correctly, and listen when the right character finally speaks. Their words, taken together, form a flag in the format LNM{...}.

What you need

  • A way to run the Shakespeare Programming Language (SPL) – for example, one of the translators linked in public SPL resources.
  • Basic understanding that SPL uses characters as integer variables and phrases like speak your mind to output ASCII.

Goal

Run bard.spl with an SPL translator and capture its output. The output string will be your flag, in the form LNM{...}.

Hint for organizers only: the script encodes the flag using integer arithmetic on characters, then reveals it via speak your mind, similar to examples in Shakespeare Programming Language demos.[page:1]
